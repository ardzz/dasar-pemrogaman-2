#### Aset Kode Lembar Kerja 1.1
```python
def input_number(message: str) -> int or float:
    return identify_number(input(message))


def identify_number(number) -> int or float:
    try:
        return int(number)
    except ValueError:
        try:
            return float(number)
        except ValueError:
            raise ValueError
        except TypeError:
            raise TypeError
```
